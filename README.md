## 项目简介

这是一个精选的 AI Agent Skills 集合，收录了来自顶级开发团队（Anthropic、Google Labs、Vercel、Stripe、Cloudflare、Netlify、Trail of Bits、Sentry、Expo、Hugging Face、Figma 等）和社区构建的真实 Skills。

**与 Claude Code、Codex、Antigravity、Gemini CLI、Cursor、GitHub Copilot 等兼容。**

## 安全提示

⚠️ **Skills 可能包含：**
- 提示词注入
- 工具投毒
- 隐藏恶意软件负载
- 不安全的数据处理模式

**使用前请：**
1. 审查代码来源
2. 验证作者
3. 使用 [Snyk Skill Security Scanner](https://github.com/snyk/agent-scan) 扫描

## 常见问题

### Q: 这些 Skills 支持中文吗？
**A:** 大部分是英文 Skills，但本仓库提供中文描述和分类，降低理解门槛。

### Q: 如何提交新 Skills？
**A:** 欢迎提交 Pull Request 到原版项目！请确保 Skills 质量过关且与 AI Agent 相关。

### Q: Skills 是免费的吗？
**A:** 大部分 Skills 是免费开源的，部分可能需要付费或订阅。

### Q: 如何验证 Skills 的安全性？
**A:** 使用 Snyk Skill Security Scanner 扫描，或手动审查代码。

### Q: 支持哪些 AI 工具？
**A:** 主要支持 Claude Code、Codex、Gemini CLI、Cursor、GitHub Copilot 等。
